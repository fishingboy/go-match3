# Go Match-3 三消遊戲

用 Go 標準庫寫成的終端機三消（Match-3）遊戲：交換相鄰寶石、連成三個以上同色即可消除，上方寶石會掉落補位並觸發連鎖加分。

## 玩法

```bash
go run .
```

- 棋盤為 8x8，欄用字母 `a`–`h`、列用數字 `1`–`8` 表示。
- 輸入兩個相鄰座標交換寶石，例如 `a1 a2`。
- 交換後若無法消除，該步會自動還原、不扣步數。
- 一次消除越多、連鎖越長，分數越高（第 n 波連鎖每顆 10×n 分）。
- 共 20 步，步數用完遊戲結束。

| 指令 | 說明 |
| --- | --- |
| `a1 a2` | 交換兩個相鄰寶石 |
| `hint` / `h` | 顯示一步可行的移動 |
| `q` | 離開遊戲 |

## 專案結構

```
main.go          終端機介面（輸入解析、畫面渲染）
game/board.go    核心邏輯：棋盤生成、交換、消除偵測、重力、連鎖、提示、洗牌
game/board_test.go 單元測試
```

## 測試

```bash
go test ./...
```

## 需求

- Go 1.25+
- 支援 ANSI 色彩的終端機（Windows Terminal、macOS Terminal、多數 Linux 終端機皆可）
